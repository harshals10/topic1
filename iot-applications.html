IoT Applications

Summary for IOT Apps

Works on sense, think, act paradigm. The output depends upon the result we get while sensing. The main features needed are web application, software, connection between devices. Example motion sensor security system, smart bulb etc.


Summary for Smart Building

The word smart refers to something self sufficient. The main features needed include task automation, data collection, acting upon the actions. Air quality control, cost, smart thermostat are some of the smart building scenarios.


Summary for Control Interface

High level features are user friendly, security, proper conditioning. The main parts needed include database, sesnors structure plan, programming. The users are mangers, maintainence, members etc.
